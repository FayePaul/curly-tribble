Commandes git

  • git init : Crée un dépot Git nommé .git dans le dossier du projet.

  • git status : Vérifie si des fichiers ont été modifiés.

  • git add "[Nom du fichier]" : Place de nouveaux fichiers sous suivi de version.

  • git commit -m "[Message]" : Enregistre les modifications dans le dépot

  • git clone "[Lien]" : Cloner un projet de github

  • git branch : Créer une branche

  • git checkout : Changer de branche

  • git merge : Fusionner des branches
